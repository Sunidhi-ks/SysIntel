# SysIntel 

 Checks System monitoring, log inspection, and network intelligence.

## Features
- Monitor CPU, Memory, Disk
- Inspect System Logs
- Analyze Network Info
- Generate full reports
- Logging enabled with rotating logs
- Schedule with Cron

## File Structure
- `ass1.py` - System Monitor
- `ass2.py` - Log Inspector
- `ass3.py` - Network Intel
- `ass4.py` - CLI Dashboard
- `logger.py` - Logging Configuration

## Usage

```bash
python3 ass4.py --option 1
